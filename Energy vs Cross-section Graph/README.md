## Energy Data Visualization Collection
# Overview

This repository presents a collection of Python scripts and plots demonstrating different data visualization techniques using Matplotlib and NumPy.\
Each visualization style represents the same dataset — experimental and observational energy-dependent values — but displayed in distinct graphical formats suitable for research, reports, or publications.\

| File Name                                                                          | Description                                                                                                                |
| ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **Graph_Plot_1.py / Graph_Plot_1.png**                                             | Standard **line plot** showing energy vs. cross-section comparison between experimental and observational data.            |
| **Graph_Plot_2.py / Graph_Plot_2.png**                                             | **Styled curve plot** with smoothed interpolation and enhanced color palette for better presentation in publications.      |
| **Graph_Plot_3_Log scale.py / Graph_Plot_3_Log scale.png**                         | Visualization in **logarithmic scale** for both axes, highlighting low-value variations and exponential trends.            |
| **Graph_Plot_4_Scatter.py / Graph_Plot_4_Scatter.png**                             | **Scatter plot** version for discrete energy points, ideal for displaying raw experimental data without interpolation.     |
| **Graph_Plot_5_Area Plot.py / Graph_Plot_5_Area Plot.png**                         | **Area plot** emphasizing integrated data regions; suitable for showing total yield or cumulative behavior.                |
| **Graph_Plot_6_Heatmap.py / Graph_Plot_6_Heatmap.png**                             | **2D heatmap visualization**, showing density or correlation between energy and corresponding observation levels.          |
| **Graph_Plot_7_3D Visualization.py / Graph_Plot_7_3D Visualization.png**           | **3D surface plot** illustrating multi-variable relationships, useful for energy spectra or parameter space visualization. |
| **Graph_Plot_8_Boxplot & Violin plot.py / Graph_Plot_8_Boxplot & Violin plot.png** | **Statistical representation** of data spread using box and violin plots to compare data distributions.                    |
